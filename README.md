# Mad-Libs-Story-Generator
Our project is a console-based Mad Libs Story Generator. The goal of our
program is to create a fun and interactive game where the user types in
different kinds of words like nouns, verbs, and adjectives. After the user
enters the words, the program will place them into a story template and print
out a funny or creative story in the console.
The program will explain the game at the beginning, ask the user for different
word types, store those words using variables, lists, and dictionaries, and
then build the final story. The user will also be able to play multiple times
if they want. This outline explains how our program will work step by step.
